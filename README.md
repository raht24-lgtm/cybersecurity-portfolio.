Cover Letter – GitHub Writeups Repository

To the GitHub community,

I am creating this repository to share my walkthroughs and writeups for challenges and labs from Hacker101, TryHackMe, and PortSwigger Web Security Academy. Every post will be available in both English and Spanish.

Why both languages? Because high-quality pentesting content in Spanish is still very scarce. I want to help change that. By providing bilingual writeups, I hope to encourage and enable more Spanish‑speakers to enter the world of ethical hacking.

These writeups reflect what I have learned so far – but pentesting is a never‑ending journey. There is always something new to discover, a different technique to master, or a failure to turn into a lesson. This repository is my way of documenting that continuous learning.

If you are just starting out or looking for clear, step‑by‑step explanations in either language, you are welcome here. Feedback, corrections, and collaborations are always appreciated.

Thank you for visiting, and remember: in pentesting, we learn something new every day.

Carta de presentación – Repositorio de Writeups en GitHub

A la comunidad de GitHub,

Creo este repositorio para compartir mis walkthroughs y writeups de desafíos y laboratorios de Hacker101, TryHackMe y PortSwigger Web Security Academy. Cada publicación estará disponible tanto en inglés como en español.

¿Por qué ambos idiomas? Porque el contenido de calidad sobre pentesting en español es aún muy escaso. Quiero contribuir a cambiar eso. Al ofrecer writeups bilingües, espero animar y facilitar que más personas hispanohablantes se inicien en el mundo de la seguridad ofensiva.

Estos writeups reflejan todo lo que he aprendido hasta ahora – pero el pentesting es un camino sin fin. Siempre hay algo nuevo por descubrir, otra técnica que dominar o un error del que aprender. Este repositorio es mi forma de documentar ese aprendizaje continuo.

Si estás empezando o buscas explicaciones claras, paso a paso, en cualquiera de los dos idiomas, eres bienvenido. Los comentarios, correcciones y colaboraciones son siempre bien recibidas.

Gracias por visitarme, y recuerda: en el pentesting se aprende algo nuevo cada día.

Rene Torres.
